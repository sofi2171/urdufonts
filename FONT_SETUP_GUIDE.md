# How to Add TTF Font Files

## Step 1: Download Urdu Fonts
Get TTF files from these sources:

### Free Urdu Fonts:
1. **Google Fonts**: 
   - Noto Nastaliq Urdu
   - Download from: https://fonts.google.com/noto/specimen/Noto+Nastaliq+Urdu

2. **Free Font Websites**:
   - Jameel Noori Nastaleeq
   - Pak Nastaleeq
   - AlQalam fonts

3. **GitHub Repositories**:
   - Search for "Urdu fonts" on GitHub
   - Many open-source Urdu fonts available

## Step 2: Replace Placeholder Files
Replace these files in `assets/fonts/` folder:

- nastaliq.ttf → Your Nastaliq font
- naskh.ttf → Your Naskh font  
- kufi.ttf → Your Kufi font
- thuluth.ttf → Your Thuluth font
- diwani.ttf → Your Diwani font
- ruqaa.ttf → Your Ruqaa font

## Step 3: File Requirements
- **Format**: .ttf or .otf
- **Size**: Keep under 5MB each
- **Names**: Use exact names as listed above

## Step 4: Test Downloads
1. Open index.html in browser
2. Click download buttons
3. Verify files download correctly
